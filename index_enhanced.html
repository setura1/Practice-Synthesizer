軽量MIDI学習シンセ v2 先にレンダリング → あとで再生

重いリアルタイム合成を避け、MIDIを先にPCMへレンダリングしてから再生します。再生中はリサジュー表示、ボード線図風の片対数FFT、短時間FFTを表示します。擬似リバーブ/ディレイによる空間系、Karplus-Strong による物理モデリングも加え、シンセサイザーとDTMの学習に使えるようにしています。

音色を切り替えると、その音色に合う初期 ADSR とローパス設定を自動で読み込みます。さらに、擬似リバーブ/ディレイの mix・feedback・density を変えると、時間方向の密度と減衰を比べられます。Karplus-Strong String は短い雑音を遅延ループで循環させ、弦をはじく近似モデルとして学べます。

学習メモ

          
    
            
  • 周波数 f = 440 × 2^((n-69)/12) でMIDI番号と音高がつながる
    • 
            
  • 矩形波やパルス波は倍音が多く、ファミコン風になりやすい
    • 
            
  • FFT は周波数軸を片対数にすると、ボード線図の見方で倍音や減衰を追いやすい
    • 
            
  • 短時間FFT は時間に沿ったスペクトル変化を見る
    • 
            
  • ピアノ風・ギター風は減衰付きの倍音合成で近似している
    • 
            
  • ADSR を変えると打鍵感と余韻が変わる
    • 
            
  • Low-pass cutoff を下げると高域が減り、音が丸くなる
    • 
            
  • 擬似リバーブ/ディレイは、遅延と帰還によって時間方向の密度を増やす
    • 
            
  • Karplus-Strong は短い雑音を遅延ループで回し、弦の減衰を離散時間で近似する
    • 
          

        

        

          
軽量化メモ

          
    
            
  • 外部ライブラリなし
    • 
            
  • レンダリング後に再生するため再生中の負荷を低減
    • 
            
  • 再生はステレオ化し、空間系とリサジュー表示に利用
    • 
            
  • ドラムチャンネルは除外
    • 
            
  • 最大同時発音数で負荷を抑制
    • 
            
  • ローパスはレンダ後に一括適用するため再生時の負荷が小さい
    • 
          

        

      

      

        
使い方

        
1. MIDIを選ぶ
2. 「MIDIを読み込んでレンダリング」を押す
3. 完了後に「再生」を押す
4. 再生中にリサジュー・片対数FFT・短時間FFTを見る
5. Space mix / Delay / Density を動かして空間系の違いを確認
6. Karplus-Strong String を選ぶと弦モデルも試せる
7. 「試しにA4を鳴らす」で音色の違いをすぐ確認

        
短時間FFTは、再生中の周波数スペクトルを左へ流すのではなく、右へスクロールしながら記録しています。現在の音の変化を目で追うための表示です。
